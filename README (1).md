# Coffee-Sales-Dashboard-using-EXCEL + POWER QUERY

**Project Overview**

This project is an interactive Coffee Sales Dashboard built using Microsoft Excel and Power Query (ETL process). It provides meaningful insights into sales performance, customer behavior, and product trends across different store locations.

The goal of this project is to demonstrate data cleaning, transformation, and visualization skills using Excel for real-world business analysis.

🎯 **Objectives**

Analyze coffee shop sales data

Identify top-performing products and categories

Understand customer footfall patterns

Track sales trends over time (monthly, daily, hourly)

Build an interactive dashboard for decision-making

🛠️ **Tools & Technologies Used**

Microsoft Excel

Power Query (ETL)

Pivot Tables

Charts & Visualizations

Slicers (for interactivity)

🔄 **ETL Process (Power Query)**

The following data transformation steps were performed:

✅ Changed data types for consistency

✅ Extracted Month, Day, Hour from date columns

✅ Created conditional columns for better categorization

✅ Removed unnecessary spaces (data cleaning)

✅ Replaced inconsistent or incorrect values

📊 **Dashboard Features**

The dashboard includes the following key insights:

💰 Total Sales

🧾 Average Order Value

👣 Total Footfall

🛍️ Average Bill

⏰ Quantity Ordered by Hour

📅 Orders by Weekdays

🥇 Top 5 Products

📍 Sales & Footfall by Store Location

🍩 Category-wise Sales Distribution

🥤 Order Size Distribution

🎛️ Interactivity

📅 Month-wise filtering

📆 Day-wise filtering

Dynamic charts update based on slicer selection

🔍 **Key Business Questions Answered**

This dashboard helps answer important business questions such as:

What is the total revenue generated?

Which month has the highest sales?

What are the peak hours for orders?

Which day of the week has the highest sales?

What are the top 5 best-selling products?

Which store location performs the best?

What is the distribution of sales across product categories?

What is the average order value?

What is the customer footfall trend?

What size of orders (small, regular, large) is most popular?

📈 **Insights (Example)**

Coffee category contributes the highest percentage of sales

Peak ordering hours are in the morning and evening

Weekdays like Friday show higher sales compared to weekends

Certain store locations outperform others in both revenue and footfall
